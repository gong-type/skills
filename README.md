# Skills 工厂

这个仓库是 skills 的生产与沉淀目录。当前以“文档处理 + 搜索 + 研发类协作”为核心，覆盖 Office 文档、代码检索、前端与移动端开发，以及系统化调试与技能生产。

## 现有 skills 简述
1. `docx`：处理 Word 文档（读取、编辑、批注、格式化、模板等）。
2. `file-search`：高效代码/文本搜索与结构化匹配（基于 ripgrep/ast-grep）。
3. `frontend-design`：生成高质量前端界面与组件，强调设计感与可落地实现。
4. `mobile-developer`：跨平台移动端开发（React Native/Flutter），偏重性能与平台特性。
5. `skill-creator`：创建/更新技能的流程与工具支持。
6. `systematic-debugging`：系统化排障方法论与操作指南。
7. `uv-python`：强制以 `uv` 管理 Python 项目与依赖的工作流。
8. `xlsx`：处理 Excel/表格文件（读写、清洗、格式化、计算等）。
